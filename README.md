# CLIP-Analysis

Использовался python 3.12

---

Как запустить:
1. Создать виртуальное окружение venv (для Windows `python3.12 -m venv windwos_env`)
    - скачать все необходимые библиотеки `pip install -r lib.txt`
3. запустить download_cifar.py - загрузка датасета, необходимого для обучения;
4. запустить main.ipynb
